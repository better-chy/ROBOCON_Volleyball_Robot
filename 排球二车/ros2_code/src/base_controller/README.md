# base_controller

Wheel order used in code:

- front_left = wheel 2
- front_right = wheel 3
- rear_left = wheel 1
- rear_right = wheel 4

Robot layout viewed from top:

- wheel 2: northwest
- wheel 3: northeast
- wheel 1: southwest
- wheel 4: southeast

Body frame:

- +x: forward
- +y: left
- +z: up
- +yaw: counterclockwise
